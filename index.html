Database Setup Instructions

Create a new table in your Supabase database by running the SQL query below in your Supabase SQL Editor:

CREATE TABLE IF NOT EXISTS kartabot_projects (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL UNIQUE,
  bot_name TEXT NOT NULL DEFAULT 'KartaBot',
  avatar_url TEXT NOT NULL DEFAULT 'https://images.unsplash.com/photo-1618005182384-a83a8bd57fbe?w=100&auto=format&fit=crop&q=80',
  welcome_message TEXT NOT NULL DEFAULT 'Hello! Welcome to our website. How can I help you today? 😊',
  n8n_url TEXT,
  suggestions TEXT,
  theme TEXT NOT NULL DEFAULT 'light',
  color TEXT NOT NULL DEFAULT '#6366f1',
  position TEXT NOT NULL DEFAULT 'right',
  created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL,
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

ALTER TABLE kartabot_projects ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow public read and write" ON kartabot_projects FOR ALL USING (true) WITH CHECK (true);

Ensure Row Level Security (RLS) is configured to allow public reads and writes as defined by the policy above for testing.
